# **App Name**: Economic Compass

## Core Features:

- Global Macro Dashboard: Track economic fundamentals like Fed rates, inflation, and recession probability.
- Crypto Market Overview: Analyze BTC/ETH cycles using log regression bands and ROI.
- DCA Simulator: Backtest dollar-cost averaging strategies with historical portfolio performance data.
- Portfolio Optimizer: Optimize crypto allocations using Sharpe ratio and efficient frontier calculations using MPT.
- Economic Alerts Feed: Provide notifications for policy shifts like rate decisions and CPI releases, as determined using a tool.
- Authentication and Profile Saving: Provide full authentication locally with Better SQLite. Let the user save the display settings

## Style Guidelines:

- Choose a dark color scheme. Background color: Dark desaturated blue (#223344). This evokes trust and seriousness appropriate for financial analysis.
- Primary color: Vibrant cyan (#00FFFF) for charts and data highlights, ensureing visibility against the dark background.
- Accent color: Analogous green (#00FF80) for positive indicators and calls to action.
- Body text: 'Inter' sans-serif, for a modern, machined look that is easily legible. Headline: 'Space Grotesk' for a techy scientific feel.
- Use minimalist icons to represent economic indicators and data sources.
- Design a clear, data-dense layout with a focus on readability and quick insights. Accommodate both Arabic and English layouts seamlessly.
- Subtle animations for data updates and user interactions to enhance the user experience without distracting from the information. Provide a toggle for disabling animations.
